Phylogenetic tree of the cryptochrome/photolyase family. Protein sequences were aligned using ClustalX [61], 2.0.12 and the phylogenetic tree was constructed with PHYLIP. For details see materials and methods. Different bootstrap values are highlighted with different grey bars.
